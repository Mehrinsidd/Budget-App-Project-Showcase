# Budget App (Showcase Version)

## Overview
The Budget App is a team-based software engineering project focused on helping users track weekly spending and manage their budget through a simple, user-friendly interface. The goal of the project was to apply real-world software engineering practices—including Agile methodologies, version control, and team collaboration—to build a fully functional budgeting tool.

This is a **showcase version** of the project.  
Full source code and deployment scripts are kept private to comply with academic integrity policies.

---

## Key Features
- User-friendly interface for entering expenses  
- Weekly budget tracking  
- Summary of total spending and remaining balance  
- Responsive UI  
- Backend routes for handling budget and expense data  
- Database integration (full version)
